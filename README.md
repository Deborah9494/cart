## Description
Simple web application to manage a warehouse using **HTML, CSS, JavaScript and jQuery**.

The page displays a table containing the items stored in the warehouse and their quantities.  
Users can add new items or update existing ones through a dynamic form.

## Features
- Display warehouse items in a table
- Add new items with name and quantity
- Update quantity if item already exists
- Set a maximum storage limit (default: 30)
- Warning message when the limit is exceeded
- Dynamic DOM manipulation with jQuery

## Technologies
- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap

## How to Run
1. Clone or download the repository
2. Open `warehouse.html` in a browser
3. Use the buttons to manage the warehouse
